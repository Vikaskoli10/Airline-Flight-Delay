# Airline-Flight-Delay

1. ✈️ Project Title / Headline

Airline Flight Delays Analysis Dashboard using Power BI

A comprehensive business intelligence solution designed to analyze airline operations at scale.
This project transforms complex flight data into meaningful insights on delays, cancellations, and performance.
It enables data-driven evaluation of airline reliability and operational efficiency.
Built to uncover disruption patterns and seasonal travel trends across the aviation network.

2. 📌 Short Description / Purpose

The objective of this project is to analyze large-scale airline operational data and convert it into actionable insights.
It helps stakeholders understand delay patterns, identify key disruption factors, evaluate airline performance, and monitor travel demand trends using interactive visualizations.

3. 🛠️ Tech Stack

Power BI 📊

DAX (Data Analysis Expressions) ⚙️

Data Modeling 🧩

Data Visualization 📈

Exploratory Data Analysis (EDA) 🔍

4. 📂 Data Source

Dataset sourced from Maven Analytics
Contains 5.8 million+ rows, providing a robust foundation for large-scale analysis

5. 🚀 Features / Highlights

Interactive dashboard with Power BI Bookmarks for seamless navigation
Analysis of flight cancellations and identification of key causes (weather, carrier issues)
Detailed delay distribution across multiple categories (carrier, weather, NAS, security, late aircraft)
Airline performance benchmarking based on delay hours and reliability
KPI tracking: % of delayed flights and average delay duration
Flight volume trends analysis by month and weekday to identify seasonality
Scalable data model handling millions of records efficiently
